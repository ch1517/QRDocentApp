## Introduce Application
```
 오프라인 도슨트 프로그램은 시간제로 운영된다는 한계가 있으며, 
 책자, 안내기기와 같은 보조품은 지원 수량의 제한이 있다.
 그러면 다른 방법이 없을까?
 
 스마트폰의 카메라를 이용한 QR 기능으로 간편하게 도슨트를 이용해보자
 ```
